# Contacts Book

A Basic Contacts Storing App using React and Redux.

## Features
- Add, Edit, Delete and Search Contacts.
- Contacts are saved in Browsers Local Storage.

## Installation of Dependencies

```
$ npm install
```

## Running App

```
$ npm run start
```
